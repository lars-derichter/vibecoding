# Vibecoding with Claude Code (for students)

A hands-on workshop introducingstudents to vibecoding with Claude Code. Covers the vibecoding workflow, practical tips, risks & limitations, a live demo, and a guided exercise where students build their own project.

Given on 2026-05-12 to Students at Thomas More University of Applied Sciences.

Visit: https://lars-derichter.github.io/vibecoding/ for the slides and download the [handout pdf](handout.pdf)

## Resources

- [Personal CLAUDE.md](resources/personal-claude-md/CLAUDE.md) — example user-wide instructions file
- [settings-example.jsonc](resources/settings-example.jsonc) — annotated `~/.claude/settings.json` template
- [atomic-commits skill](resources/skills/atomic-commits.md) — skill for splitting messy changes into clean commits
