# Vibecoding with Claude Code (for Business students)

A hands-on workshop introducing Business Students to vibecoding with Claude Code. Covers the vibecoding workflow, practical tips, risks & limitations, a live demo, and a guided exercise where students build their own project.

Given on 2026-03-26 to Business Students at Thomas More University of Applied Sciences.
